## GroupCard

A full-stack prototype for group payments that splits a single merchant charge into
multiple user holds with transactional guarantees.

### Tech Stack
- Next.js (App Router)
- TypeScript
- Node.js
- PostgreSQL / Supabase

### Key Features
- Multi-party payment flow with all-or-nothing guarantees
- State machine: authorize → confirm → split → capture
- REST APIs for payment orchestration
